# GPSR-003 Investment Ontology

## Canonical Hierarchy
Reality -> Signals -> Events -> Themes -> Megatrends -> Activation Contexts -> Persistent World Graphs -> Activated Nodes -> Constraints -> Bottlenecks -> Economic Profit Pools -> Companies -> Institutional Capital -> Stock Prices

## Core Entities
- **Reality**: Objective changes in the world independent of markets.
- **Signal**: Atomic observation extracted from data.
- **Event**: One or more related signals forming a meaningful occurrence.
- **Theme**: A coherent emerging pattern created by related events.
- **Megatrend**: Multiple reinforcing themes producing long-term structural change.
- **Activation Context**: The set of themes, macro conditions and catalysts that activate portions of persistent graphs.
- **Persistent World Graph**: A long-lived graph representing an enduring economic system.
- **Value Chain**: Graph describing how value is created.
- **Supply Chain**: Graph describing how products and capabilities flow.
- **Constraint**: Condition limiting growth or production.
- **Bottleneck**: Constraint that becomes economically scarce.
- **Economic Profit Pool**: Location where excess returns accrue.
- **Company**: Public or private participant in one or more graphs.
- **Knowledge Object**: Validated scientific knowledge published by Genesis.
- **Historical World Graph**: Historical snapshot of an evolving economic system.
- **Hypothesis**: Testable explanation awaiting validation.
- **Evidence**: Observation supporting or contradicting a hypothesis.
- **Investment Thesis**: Reasoned capital allocation proposal.
