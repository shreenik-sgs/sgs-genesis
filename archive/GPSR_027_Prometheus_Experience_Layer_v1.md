# GPSR-027 Prometheus Experience Layer

Purpose: Operational memory for the live investment intelligence system.

## Stored Objects
- Current World State Graph
- Active Signals
- Active Events
- Active Themes
- Activated Persistent World Graphs
- Constraint Register
- Bottleneck Register
- Value Capture Graphs
- Candidate Companies
- Leadership Assessments
- Technical Assessments
- Capital Allocation Plans
- Exit Decisions
- Alpha Checkpoints
- Prediction History
- Portfolio Outcome History

## Lifecycle
1. Receive new intelligence
2. Update operational objects
3. Version material changes
4. Notify dependent engines
5. Record user actions
6. Capture outcomes
7. Generate Alpha Checkpoints
8. Forward learning candidates to Recursive Self-Improvement
