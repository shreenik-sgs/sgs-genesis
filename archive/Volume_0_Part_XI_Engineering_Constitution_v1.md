# Volume 0 - Part XI: Engineering Constitution

## MUST
- All engines MUST consume and emit canonical graph objects.
- Every recommendation MUST reference supporting Economic Laws or Knowledge Objects.
- Every graph mutation MUST preserve provenance.
- All APIs MUST be versioned.
- Historical replay MUST reproduce prior graph state.
- All permanent knowledge MUST pass Scientific Council governance.
- Every production release MUST pass historical replay and regression tests.

## SHOULD
- Services SHOULD be stateless where practical.
- Graph transformations SHOULD be deterministic.
- Long-running workflows SHOULD be event-driven.
- Interfaces SHOULD remain backward compatible.
- Every module SHOULD expose health and telemetry endpoints.

## MAY
- Implement distributed graph execution.
- Support multiple graph databases.
- Introduce Bayesian confidence propagation.
- Use hardware acceleration for graph analytics.
- Deploy specialized agent clusters.
