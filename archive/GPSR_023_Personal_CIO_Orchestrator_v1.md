# GPSR-023 Personal CIO Orchestrator

Purpose: Executive orchestration layer connecting Genesis, Prometheus, and the investor.

## User Interaction Model
1. Morning briefing summarizing world changes and portfolio impact.
2. Review recommended buys, adds, trims, and exits.
3. Inspect explainable reasoning through the Infinite Canvas.
4. Approve, reject, or defer actions.
5. Capture user feedback for Recursive Self-Improvement.
6. Monitor intraday alerts and thesis changes.
