# GPSR-016 Value Capture Engine

Purpose: Determine where economic profits accumulate before selecting companies.

## Value Capture Dimensions
- Pricing power
- Supply concentration
- Technology differentiation
- Switching costs
- Replacement difficulty
- Capital intensity
- Intellectual property
- Network effects
- Scale advantage
- Regulatory protection
- Execution capability
- Leadership quality

## Processing Pipeline
1. Load bottlenecks
2. Estimate affected profit pools
3. Propagate economics through value graph
4. Estimate capture by each value layer
5. Compare against historical analogues
6. Publish Value Capture Graph
7. Forward candidates to Company Discovery
